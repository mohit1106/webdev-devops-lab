creating a nextJs app and whenever pull requests come do this:
    - When a push happens on the main branch
    - Run a ci/cd script that will ssh into ec2 machine
    - npm run build inside of ec2
    - npm run start inside of ec2

    create a folder in repository codebase:
        .github/workflows/deploy.yml

        //////////////  https://github.com/actions/checkout/tree/v2 /////////////////////////////
        //////////////  https://github.com/appleboy/ssh-action /////////////////////////////
        //////////// add this in deploy.yml ////////////////////////////////////////////////////////////////////////////////
        name: Deploy to EC2

        on:
            push:
                branches:
                    - main

        jobs:
            deploy:
                name: Push to EC2
                runs-on: ubuntu-latest

                steps:
                    - name: Checkout the code
                      uses: actions/checkout@v2

                    - name: Execute remote SSH commands using password
                      uses: appleboy/ssh-action@v1
                      with:
                        host: ${{ secrets.HOST }}
                        username: linuxserver.io
                        password: ${{ secrets.PASSWORD }}
                        port: ${{ secrets.PORT }}
                        script: whoami
