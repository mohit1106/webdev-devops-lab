now go to publicip:port   and its deployed, its not in https but in http

http defaault port is 80,  so, http://publicip === http://publicip:80
https defaault port is 443,  so, https://publicip === https://publicip:443
so that we dont need to specify port in the url, but we cant run anything on 80 or 443, also only one service can run on it.
so we use NGINX for reverse proxing.
